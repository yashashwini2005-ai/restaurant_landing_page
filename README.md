# Aurelia Fine Dining Landing Page

A modern, premium, and fully responsive restaurant landing page designed for a luxury dining experience.

## Features

* Responsive Design (Mobile, Tablet, Desktop)
* Elegant Hero Section
* About Us Section
* Featured Dishes Showcase
* Customer Testimonials
* Table Reservation Form
* Smooth Scroll Animations
* Interactive Navigation
* Modern UI/UX Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
